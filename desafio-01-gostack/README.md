##  Sobre o desafio

Criando uma aplicação para armazenar projetos e suas tarefas do zero utilizando [Express](https://expressjs.com/pt-br/).

### Rotas

- `POST /projects` 

- `GET /projects`

- `PUT /projects/:id`

- `DELETE /projects/:id`

- `POST /projects/:id/tasks`

### Exemplo

```js
[
  {
    id: "1",
    title: "BOOTCAMP",
    tasks: ["Desafio 01", "Desafio 02", "Desafio 03", "Desafio 08"]
  }
];
```
